# Shard of the Triforce

Browser-playable 2D Zelda-like action-adventure (Rust → WASM, Canvas2D,
"Minish Cap cleanline" art direction). Act 1 polished first; Acts 2–3 follow.

All decisions, architecture, design, and phase plans live in `docs/`:

- `docs/DECISIONS.md` — locked stack/art/audio/input/save/deploy decisions (source of truth)
- `docs/ARCHITECTURE.md` — crate layout, rendering, entity model, ownership boundaries
- `docs/GAME_DESIGN.md` — Act 1 design: combat kit, overworld, dungeon, Granite Warden
- `docs/PHASE_PLAN.md` — phases 0–5 with acceptance criteria
- `docs/WORKER_BRIEF_PHASE0.md` — current executable worker brief

Run instructions land with Phase 0 (Rust scaffold).
